************
Installation
************

Requirements
============

::

   Asterisk-13.6 or later
   Asterisk-14.3 or later
   Asterisk-manager module
   libsqlite3
   libevent2
   libbsd
   libjansson
   libevhtp
   libssl

Install
=======

::

   $ sudo mkdir -p /opt/bin
   $ cd src/
   $ make
   $ sudo mv jade_backend /opt/bin
